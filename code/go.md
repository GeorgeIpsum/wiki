# go

## Installing go

### On macOS

Use homebrew:

```bash
brew update
brew install golang
```

### On Windows

Use chocolatey:

```bash
choco install -y golang
```

### On Linux

#### Deb

```bash
sudo apt update
sudo apt install golang-go
```

#### RPM

Install it from source basically.

```bash
sudo yum update
wget https://dl.google.com/go/go1.13.1.linux-amd64.tar.gz
tar -xzf go1.13.1.linux-amd64.tar.gz
mv go /usr/local
```
